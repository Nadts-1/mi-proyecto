Programa para que verifiques la mayoria de edad

En el código se ve, que se solocita la edad de un usuario para ver si es mayor o menor de edad


#include <stdio.h>
int main()
{
    int edad;
    printf("¿Cuántos años tienes? ");
    scanf("%d", &edad);


    if (edad >= 18) 
    {
        printf("Eres mayor de edad.\n");
    } 
    else 
    {
        printf("Eres menor de edad.\n");
    }


    return 0;

}

Explicación de cada parte del Código

#include <stdio.h> Es la libreria que se ocupa para que el programa en C puedan funcionar

int main() Es la función principal del programa todo lo que este adentro de sus llaves es lo que va a leer la computadora y lo va a ejecutar.

int edad; Es para declara una variable entera. Es como crear un espacio en la memoria para guaradar el no. de años que tiene la persona

printf Es una función que sirve para mostrar un mensaje en la pantalla y que lo pueda ver el usuario

scanf Es para leer lo que el usuario escribe. El "%d" le dice que va a recivir un no. entero y el &edad lo guarda dentro de la variasble edad

Estructura if. Es una condicion que sirve para evaluar si el valor guardado en "edad" es mayor o igual a 18 y si es verdadereo se muestra el mensaje

Estructura ELSE. Es como lo negativo si el if no se cumple, el programa no entra al if y entra al else y muestra otro mensaje

return 0; Es para avisarler al sistema que el programa se ejecuto sin errorres y correctamente
